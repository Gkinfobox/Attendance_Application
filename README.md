# 📍 Employee Attendance App

An Android application for location-based User attendance tracking.

## Features
- 🔐 Secure user login via Firebase Authentication
- 📍 Location-based check-in system
- 🏠 Real-time check-in status on home screen
- 📊 10-day attendance summary
- ☁️ Data stored in Google Sheets via API
- 🔥 User records managed through Firebase

## Tech Stack
- Java + Android Studio
- Firebase Authentication & Realtime Database
- Google Sheets API
- Location Services (GPS)

## Status
✅ Production deployed — actively used by a real company
